#  Password Strength Checker

## Описание
Программа проверяет пароли по нескольким критериям:
- Длина пароля
- Наличие заглавных и строчных букв
- Наличие цифр и специальных символов
- Проверка на слабые пароли

Также умеет генерировать надежные пароли и сохранять результаты проверки.

##  Как запустить
1. Установи Python 3.x
2. Скачай файл 'password_checker.py'
3. Запусти в терминале
